How Forums Work

  • Cryptographic Integrity - Every post is signed with ML-DSA-87 (post-quantum secure)
  • DAG Structure - Content forms a Directed Acyclic Graph that can be rebuilt from any backup
  • Content Addressing - Each node is identified by its SHA3-512 hash
  • Moderation - Forum owners can appoint moderators who can create boards
  • Public - All content is public and readable by anyone
